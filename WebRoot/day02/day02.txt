CSS
	cascading style sheet
	层叠样式表
	
	为html等页面添加外观和样式
	
	页面的内容,即数据,由html实现的
	页面的效果,即外观,由css实现
	页面的行为,即功能,由javaScript实现的		
		
	将页面的内容和效果分开,便于维护		
		
	选择器:
		标签选择器:
		标签名{
			属性名:属性值;
			属性名:属性值;
			属性名:属性值;
			属性名:属性值;
			......
		}
		
		id选择器:
		#选择器名称{
			
		}
		
		类选择器(class)
			.选择器名称{
			
				
			}
	
		混合选择器
		选择器1,选择器2...{
			....
		}
		使用逗号分隔,表示这些选择器都使用该样式
		
		选择器1 选择器2{
			....
		}
		以空格分隔,表示选择器1下面的选择器2使用该样式
	
	
	
	三种选择器的优先级:
		id选择器>类选择器>标签选择器
	
		
	css样式继承
		若子标签没有设置样式,那么子标签会继承父标签的样式
			
	
	
	
	css样式的三种引入方式:
		行内式:
			<span style="color: yellow">这是div外面的span</span>
			
		内联样式:
			 <style type="text/css">
	    	</style>		
		
		外部引入:
			<link rel="stylesheet" href="style.css" type="text/css"></link>
			
		三种方式的优先级:	
			行内式>内联样式>外部引入
			
		有一种情况可以破坏优先级
		!important
		使用!important关键字后该样式优先级最高
		
	css总体优先级:
		行内式>id选择器>类选择器>标签选择器

	color:颜色
		计算机中有两种颜色体系:RGB、CMYK
		显示器使用的RGB色系,取值0-255(换算成16进制00-ff)
		00表示颜色最浅,ff表示颜色最深
		注:这边的深表示偏向该颜色,浅表示偏向黑色
		#RRGGBB
			RR表示红色的深浅度
			GG表示绿色的深浅度
			BB表示蓝色的深浅度
		
	常用的css属性
		position:将元素放置到一个静态的、相对的、绝对的、固定的位置中
		
		属性值:
			static:默认值,静态定位,是直接按照浏览器的顺序定位
					从左往右,从上往下
			absolute:绝对定位
				参照父元素偏移
				元素在文档刘中完全删除,并相对于其包含块定位
				绝对定位在做偏移的时候,相对的是其祖先元素,而不是其本身
	
				简单来说,如果文档中某个元素设置了绝对定位,那么该文档中其他元素进行定位的时候
				将会无视设置了绝对定位的元素,当该元素不存在
	
			relative:相对定位
				首先按照默认方式排放,再做偏移
				元素偏移某个距离,相对的是其本身的位置
		
		
			总体来说,绝对定位是相对于浏览器,相对定位是相对于自己本身所处的位置
				
		偏移属性:
			top:元素的上边距距离父元素的距离的上边框的距离
			bottom:元素的下边距距离父元素的下边框的距离
			left:元素的左边框距离父元素的左边框的距离
			right:元素的右边框距离父元素的右边框的距离
			
		
		
		display
			组件的显示方式	
			属性值有:
				inline:将元素按照行级元素显示,前后没有换行符
				block:将元素按照块级元素显示,前后带有换行符
				none:不显示内容,隐藏元素
			
		
		float:
			浮动
			当空间足够的时候,设置浮动,元素会自动跑到紧贴上一个元素的位置,
			当空间不足的时候,元素会换行
			
		
		
		盒子模型
			元素在html页面中所占用的位置称之为盒子模型(框模型)
			
			
			border:边框
				style:边框的类型
					当属性值只有1的时候,表示容器的4个边框都是该类型
					当属性值有2个的时候,表示上下是第一种类型,左右是第二种类型
					当属性值有3个的时候,分别表示上、左右、下
					当属性值有4个的时候,分别表示上、右、下、左
					下面的属性值与之类似
				color:边框的颜色
				width:边框的宽度
			
			margin:外边距
				容器与外界屏幕的距离
				当属性值只有1的时候,表示容器的4个边框都是该类型
				当属性值有2个的时候,表示上下是第一种类型,左右是第二种类型
				当属性值有3个的时候,分别表示上、左右、下
				当属性值有4个的时候,分别表示上、右、下、左
			padding:内边距
				容器内部与容器的距离
				当属性值只有1的时候,表示容器的4个边框都是该类型
				当属性值有2个的时候,表示上下是第一种类型,左右是第二种类型
				当属性值有3个的时候,分别表示上、左右、下
				当属性值有4个的时候,分别表示上、右、下、左
